### Q1: 關於 AI，下列敘述何者正確？

- (A) AI 僅能處理結構化數據的分析
- <span style='color: #0066cc; font-weight: bold;'>(B) AI 涵蓋多種專業領域與技術</span>
- (C) AI 系統只能在學術研究中應用
- (D) AI 無法應用於金融領域

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(B) AI 涵蓋多種專業領域與技術</span>  
<span style='color: #646464;'>💡 為什麼？ AI 是一門廣泛的學科，包含機器學習、自然語言處理、電腦視覺等多個領域，且已廣泛應用於醫療、金融、製造等各種實務產業及非結構化數據處理中。</span>

### Q2: 下列何者最適合訓練電腦下圍棋、自動駕駛等動態重複地互動的問題？

- (A) 監督式學習（Supervised Learning）
- (B) 非監督式學習（Unsupervised Learning）
- (C) 半監督式學習（Semi-supervised Learning）
- <span style='color: #0066cc; font-weight: bold;'>(D) 強化學習（Reinforcement Learning）</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) 強化學習（Reinforcement Learning）</span>  
<span style='color: #646464;'>💡 為什麼？ 強化學習透過「代理人」與「環境」互動，根據行動後的獎勵或懲罰來學習最佳決策路徑，非常適合下棋或自動駕駛等需要連續決策的動態問題。</span>

### Q3: 深度學習模型中 ， 下列哪一項通常用來降低過擬合（Overfitting）問題？

- (A) 增加訓練數據量
- (B) 增加模型的複雜度
- (C) 增加學習率
- <span style='color: #0066cc; font-weight: bold;'>(D) 增加正則化項</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) 增加正則化項</span>  
<span style='color: #646464;'>💡 為什麼？ 正則化（如 L1 或 L2）透過在損失函數中加入對權重的懲罰，避免模型過度依賴訓練數據中的雜訊或特定細節，從而提升泛化能力並降低過擬合。</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) 以上皆是</span>  
<span style='color: #646464;'>💡 為什麼？ AI 的影響跨越國界，國際合作能建立統一的倫理與技術標準、防範跨國性的技術濫用風險，並協助各國在技術發展上取得平衡。</span>

### Q5: 若要設計一個能夠辨識並過濾垃圾郵件的系統，應該選擇下列哪一種機 器學習演算法以實現最佳效果？

- <span style='color: #0066cc; font-weight: bold;'>(A) 監督式學習（Supervised Learning）</span>
- (B) 非監督式學習（Unsupervised Learning）
- (C) 半監督式學習（Semi-supervised Learning）
- (D) 強化學習（Reinforcement Learning）

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(A) 監督式學習（Supervised Learning）</span>  
<span style='color: #646464;'>💡 為什麼？ 垃圾郵件過濾需要依靠「已標記」的歷史郵件（是或不是垃圾郵件）來訓練模型，這種利用「輸入特徵」對應「已知標籤」的學習方式屬於監督式學習。</span>

### Q7: 下列何者非大數據時代資料的特性？

- (A) 資料量大
- (B) 資料變動速度快
- (C) 資料多樣性
- <span style='color: #0066cc; font-weight: bold;'>(D) 資料存儲位置固定</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) 資料存儲位置固定</span>  
<span style='color: #646464;'>💡 為什麼？ 大數據的 5V 特性包含 Volume (量大)、Velocity (速快)、Variety (多樣)、Value (價值) 與 Veracity (真實)。「存儲位置固定」並非其特性，現代大數據通常分散存儲於雲端或分佈式系統。</span>

### Q8: 關於 K 平均法（K-means） ，下列敘述何者「不」正確？

- (A) 希望找出 k 個互不交集的群集
- (B) 不同的起始群集中心，可能會造成不同的分群結果
- (C) 容易受雜訊與離群值（Outlier）影響其群集中心
- <span style='color: #0066cc; font-weight: bold;'>(D) 可以處理類別型資料</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) 可以處理類別型資料</span>  
<span style='color: #646464;'>💡 為什麼？ K-means 演算法依賴計算樣本間的「歐式距離」來分群，因此僅適用於數值型資料，無法直接處理像「性別、地區」等類別型資料。</span>

<span style='color: #646464;'>💡 為什麼？ 決策樹以樹狀結構呈現分類邏輯，人類可以輕易閱讀並理解其分類路徑（如：若年齡 > 30 且 收入 > 5萬，則為 A 類），因此具有極高且易懂的可解釋性。</span>

### Q11: 關於「過擬合（Overfitting）」，下列敘述何者正確？

- (A) 模型在訓練數據與測試數據上的表現都很好
- (B) 模型在訓練數據上表現差，在測試數據上表現好
- <span style='color: #0066cc; font-weight: bold;'>(C) 模型在訓練數據上表現極佳，但在未見過的測試數據上表現極差</span>
- (D) 模型根本無法學習到數據中的規律

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(C) 模型在訓練數據上表現極佳，但在未見過的測試數據上表現極差</span>  
<span style='color: #646464;'>💡 為什麼？ 過擬合是指模型「背下了」訓練集中的細節與雜訊，導致它缺乏「泛化能力」，一遇到新資料就出錯，像是一個只會背題庫但不會活用觀念的學生。</span>

### Q12: 數據預處理階段進行「歸一化/正規化（Normalization）」的目的為何？

- (A) 增加資料的多樣性
- <span style='color: #0066cc; font-weight: bold;'>(B) 消除不同特徵間因數值尺度差異導致的負面影響</span>
- (C) 減少數據的樣本數量
- (D) 將非結構化資料轉為結構化資料

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(B) 消除不同特徵間因數值尺度差異導致的負面影響</span>  
<span style='color: #646464;'>💡 為什麼？ 若特徵 A 範圍是 0-1 而特徵 B 是 0-10000，模型可能會過於重視數值大的特徵 B。歸一化能將數據縮放至相同區間（如 0 到 1），確保每個特徵在模型中能公平地被考慮。</span>

### Q13: 下列哪像不是深度學習常用的啟動函數（Activation Function）？

- (A) Sigmoid
- (B) ReLU
- (C) Tanh
- <span style='color: #0066cc; font-weight: bold;'>(D) K-Map</span>

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(D) K-Map</span>  
<span style='color: #646464;'>💡 為什麼？ Sigmoid, ReLU 與 Tanh 皆為神經網路中將線性轉化為非線性的啟動函數；而 K-Map (卡諾圖) 是數位電路邏輯設計中用於化簡布林代數的工具，與深度學習無關。</span>

### Q14: 關於隨機森林（Random Forest），下列敘述何者正確？

- <span style='color: #0066cc; font-weight: bold;'>(A) 它由多棵決策樹組成，並投票得出結果</span>
- (B) 它比單一決策樹更容易產生過擬合
- (C) 它僅能用於分類，不能用於迴歸
- (D) 它是非監督式學習模型

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(A) 它由多棵決策樹組成，並投票得出結果</span>  
<span style='color: #646464;'>💡 為什麼？ 隨機森林是一種「集成學習 (Ensemble Learning)」，透過組合多棵獨立訓練的決策樹，並以「少數服從多數」或「平均值」來預測，能有效降低單一模型帶來的過擬合風險。</span>

- <span style='color: #0066cc; font-weight: bold;'>(C) 注意力機制 (Attention Mechanism)</span>

<span style='color: #646464;'>💡 為什麼？ 注意力機制讓模型在處理一串文字時，能學會給予「重要單字」更高的權重，進而精確捕捉長距離的語意關聯，這是當代 Transformer 模型（如 GPT）的核心。</span>

### Q16: 在處理資料缺失值時，若樣本數量充足，最簡單且常用的處理方式為何？

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(A) 刪除帶有缺失值的整列紀錄</span>

### Q24: 下列哪像屬於「非監督式學習 (Unsupervised Learning)」的應用？

- (A) 預測房價

- (B) AI 生成速度太快
- <span style='color: #0066cc; font-weight: bold;'>(C) 在訓練時是否未經許可使用了受版權保護的作品</span>
- (D) 模型檔案太大無法下載

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(C) 在訓練時是否未經許可使用了受版權保護的作品</span>  
<span style='color: #646464;'>💡 為什麼？ AI 的強大來自於「看過全世界的畫作/文章」。如果這些資料庫未經原創作者同意即被拿去餵養 AI，且 AI 產出的結果與原創者高度相似甚至直接模仿，就會引發法律與倫理上關於合理使用的重大抗戰。</span>

### Q49: 關於「數據清洗 (Data Cleaning)」，其最重要的目標是？

- (A) 讓資料看起來格式漂亮
- (B) 把所有數值都調高
- <span style='color: #0066cc; font-weight: bold;'>(C) 剔除錯誤、重複與錯誤格式的資料，提升數據品質 (Garbage in, Garbage out)</span>
- (D) 讓資料檔案變大以便存放

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(C) 剔除錯誤、重複與錯誤格式的資料，提升數據品質 (Garbage in, Garbage out)</span>  
<span style='color: #646464;'>💡 為什麼？ 數據清洗是 AI 的基石。如果餵給 AI 的資料充滿了打字錯誤、自相矛盾或冗餘垃圾，訓練出來的模型也會跟著出錯。這過程是確保模型輸出可信任的關鍵保障。</span>

### Q50: 在設計對話機器人時，若希望它能呈現特定的人格特質（如：像個親切的理財專員），應在哪個環節處理？

- (A) 增加硬體記憶體
- (B) 更換成更大量的 JSON 檔案
- <span style='color: #0066cc; font-weight: bold;'>(C) 透過「提示詞工程 (System Prompt Design)」設定角色引導</span>
- (D) 修改電腦作業系統時間

<span style='color: #008000; font-weight: bold;'>✅ 正確解答：(C) 透過「提示詞工程 (System Prompt Design)」設定角色引導</span>  
<span style='color: #646464;'>💡 為什麼？ 透過預先餵給模型一段「系統指令 (System Instruction)」，可以框定它的輸出風格、專業知識邊界與互動禮儀，讓 LLM 能在同一個底層模型上發展出成千上萬種不同的專家化身。</span>
